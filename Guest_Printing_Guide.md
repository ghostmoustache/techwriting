# Print Instructions for Guest Users

## Overview
This guide explains how non-affiliated users can print and copy documents in the university library.

## Print a Document

1.	Log in to any of the Mac computers using this word as both the username and password: bgsu1969 
2.	When you go to the print options in whichever software you’re using, the printer will default to BG-Print-LPR. When you select the Print button, you’ll see a window pop up requesting a username and password.
3.	Enter the long number on the back of the card as the username. Enter the word ‘guest’ in lowercase as the password, then select OK.
4.	A window will pop-up, saying that your print request has been sent to the printer. Select the OK button.
5.	Go to any of the publicly available printer/copiers and swipe your card in the black slot on the printer.
6.	Select the ‘Print Release’ option. You’ll see all of your print jobs, and you’ll be able to select which ones to print by checking either the ‘Select all’ box, or the box next to each individual print job. Once selected, click on the green Print button in the lower right corner.
7.	When you’re finished, select the log out button, next to the Home button in the bottom left corner. 

## Pricing

- Black & white: $0.05/page  
- Color: $0.20/page
  
You can check the available balance on your card by selecting the Account Selection option from the Home screen. To access the Home screen at any point, click the Home button in the bottom left corner.
